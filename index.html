FreeWheel-PHP-Wrapper by phpfunk

A PHP wrapper for the FreeWheel API.

This project is an OO PHP wrapper for FreeWheel.tv's API for their ad server. You must be a client of FreeWheel in order to use this library since you need an API key. This library will only work with PHP 5+ and works best with autoload(). Each API section is broken down into its own library extending the main library of FreeWheel. This way you don't have to load the entire library when you just need to work with just one specific API. Maintained and Created by the Viddler Team
